# Share structure (template)

> Generic Québec multi-class structure. Company-specific details belong in local minute book / Supabase, not in this public repo.

## Authorized classes (example: 7)

| Class | Type | Voting | Notes |
|-------|------|--------|-------|
| **A** | Ordinary | Yes | Residual |
| **B** | Ordinary | No | Residual (with A) |
| **C–G** | Preferred | Varies | Rollover / control / financing — see minute book |

## Agent notes

- **Salary** (owner-employee) ≠ **dividends** on ordinary shares.
- Before recommending dividends or share redemptions, flag for CPA review.
- Cap table and issuance details: minute book only (local).
